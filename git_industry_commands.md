## git add

**Syntax:**
git add <file>

**Purpose:**
Stages a file to be included in the next commit.

**Example:**
git add demo.txt

## git add -p

**Syntax:**
git add -p <file>

**Purpose:**
Stages changes in parts (hunks) interactively.

**Example:**
git add -p demo.txt

## git restore

**Syntax:**
git restore <file>

**Purpose:**
Restores a file to its last committed state.

**Example:**
git restore demo.txt

## git commit -m

**Syntax:**
