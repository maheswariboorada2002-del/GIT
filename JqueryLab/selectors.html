Quick Reference
SelectorDescription
$("p")All <p> elements
$("#id")Element with specific ID
$(".cls")All elements with class
$("a,b")All a and b elements
$("div p")All <p> inside <div>
$("ul>li")Direct <li> children of <ul>
$("h+p")First <p> after <h>
$("h~p")All <p> siblings after <h>
$("li:first")First <li>
$("li:eq(2)")3rd <li> (0-indexed)
$("li:even")Even-indexed <li> elements
$("[href]")Elements with href attribute
$("[type=text]")Elements where type equals text
